# Simulador NK para el análisis de shocks bajo distintas reglas de política monetaria

Dinero, Crédito y Bancos · UNS

Aplicación Streamlit para comparar equilibrio inicial vs. shock en un modelo Nuevo Keynesiano simple OA–RPM, bajo Regla de Taylor y Regla Monetaria Óptima.

## Uso sugerido

1. Elegir un escenario discreto: equilibrio inicial, shock transitorio de demanda, shock transitorio de oferta, aumento permanente de G, shock permanente de productividad, caída de expectativas o baja de meta.
2. Elegir la regla de política monetaria: Taylor o RMO.
3. Comparar curvas punteadas iniciales con curvas sólidas finales.
4. Usar “Ver desplazamiento paso a paso” para mostrar la transición visual entre E0 y E1.
5. Abrir “Parámetros avanzados” solo para explorar cambios de pendiente y sensibilidad.
