# Simulador NK · OA–RPM

App Streamlit para trabajar el modelo Nuevo Keynesiano simple en el plano OA–RPM.

## Archivos

- `app.py`: aplicación principal.
- `requirements.txt`: dependencias.
- `README.md`: guía breve.

## Uso en Streamlit Cloud

1. Subir estos archivos a un repositorio de GitHub.
2. En Streamlit Community Cloud, elegir el repo.
3. Como archivo principal, indicar:

```text
app.py
```

## Incluye

- Regla de Taylor y Regla Monetaria Óptima.
- Shocks transitorios de demanda y oferta.
- Shock permanente de gasto público / demanda autónoma.
- Shock permanente de producto potencial.
- Cambios en expectativas y en meta de inflación.
- Gráfico OA–RPM con escala fija y curvas iniciales de referencia.
